## The tlp's ` Mark ` Element
Mark element is used to create a interactive content writing with ability to include colors to text and serveral text editing marks 
the following are the markups used in tlp.

### tlp:codes
code start

### tlp:codee
code end

### tlp:tab

### tlp:br 

### tlp:red
